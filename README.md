## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* Python

### Installation

1. Clone the repo
```sh
git clone https://github.com/th-ms/EasyACO.git
```
2. Pip install from requirements
3. Run the app
```sh
python app.py
```
